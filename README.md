# Claude_Web
把Claude装进自己的Web模板中
## 运行
### 1.下载源码
### 2.安装库
```bash
pip install -r requirements.txt
```
### 3.修改HTML模板
亮白模式：gpt.html
暗黑模式：black_g.html
### 4.运行
端口填一个可用的即可，不填默认8080
```bash
python3 server.py 端口
```
nohup
```bash
nohup python3 server.py 端口
```
